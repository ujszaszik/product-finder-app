Can be run with latest stable Android Studio version, which is Chipmunk 2021.2.1. (May 2022)

Gradle Plugin version: 7.1.2,

Kotlin Plugin version: 1.6.10

Technologies I used:

- Jetpack Compose
- Hilt
- Coroutines / Flow
- Ktor / KotlinX
- Kotlin DSL
- Coil
- Lottie
- Timber
